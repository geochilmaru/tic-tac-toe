# Tic Tac Toe

브라우저에서 즐기는 실시간 멀티플레이어 틱택토 — 두 명의 플레이어, 하나의 서버, 프레임워크 없음.

## 게임 방법

1. 게임을 열고 **Find Game** 버튼을 클릭해 매칭 대기열에 진입합니다.
2. 두 번째 플레이어가 접속하면 게임이 자동으로 시작됩니다. X 또는 O가 무작위로 배정되며, X가 항상 먼저 시작합니다.
3. 내 차례에 빈 셀을 클릭해 말을 놓습니다. 상대방이 생각하는 동안 보드는 잠깁니다.
4. 가로, 세로, 대각선 중 하나로 세 칸을 먼저 채운 플레이어가 승리합니다. 아홉 칸이 모두 채워졌는데 승자가 없으면 무승부입니다.
5. 게임 종료 후 **Play Again** 버튼을 클릭하면 새 상대를 찾습니다.

## 기술 스택

- **런타임** — Node.js
- **서버** — Express 5
- **실시간 통신** — Socket.io
- **프론트엔드** — 바닐라 HTML / CSS / JavaScript (프레임워크 없음)

## 로컬 실행

```bash
git clone https://github.com/geochilmaru/tic-tac-toe.git
cd tic-tac-toe
npm install
node server.js
```

**http://localhost:3000** 을 두 개의 브라우저 탭에서 열면 혼자서도 테스트할 수 있습니다.

## 동작 원리

서버가 모든 게임 상태(보드, 턴 순서, 승리 판정)를 관리합니다. 클라이언트는 자체적으로 상태를 신뢰하지 않습니다.

- 플레이어가 셀을 클릭하면 클라이언트가 WebSocket으로 `make-move` 이벤트를 서버에 전송합니다.
- 서버는 이동의 유효성(올바른 플레이어, 빈 셀, 게임 진행 중 여부)을 검증한 뒤, 유효하면 `move-made` 이벤트로 양쪽 플레이어에게 업데이트된 보드를 브로드캐스트합니다.
- 매칭은 메모리 내 큐로 처리됩니다. 먼저 `find-game`을 보낸 소켓이 대기하고, 두 번째 소켓이 접속하면 즉시 매칭되어 양쪽에 `game-start` 이벤트와 배정된 심볼이 전달됩니다.
- 게임 도중 상대방 연결이 끊기면 남은 플레이어에게 `opponent-left` 이벤트가 전송되고 재경기 버튼이 표시됩니다.
